# 📊 AmbitionBox Company Data Analysis

This project focuses on fetching, cleaning, analyzing, and visualizing company data from AmbitionBox using Python. The goal is to perform Exploratory Data Analysis (EDA) and generate meaningful insights from real-world company information.

## 🚀 Features

- Fetch company data from AmbitionBox
- Data cleaning and preprocessing
- Exploratory Data Analysis (EDA)
- Rating analysis
- Company type analysis
- Location-wise company distribution
- Data visualization using charts and graphs

## 🛠️ Tech Stack

- Python
- Pandas
- NumPy
- Matplotlib
- Requests
- BeautifulSoup (if web scraping is used)
- Jupyter Notebook

## 📈 Analysis Performed

- Top-rated companies
- Distribution of companies by location
- Company type comparison
- Rating trends
- Industry-wise insights
- Data visualization with bar charts and pie charts

## ▶️ How to Run

1. Clone the repository

```bash
git clone <repository-link>
```

2. Install dependencies

```bash
pip install -r requirements.txt
```

3. Open Jupyter Notebook

```bash
jupyter notebook
```

4. Run the analysis notebook.
